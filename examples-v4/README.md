# Examples (Tailwind CSS v4)

Tailwind CSS v4 を使用したサンプルプロジェクトです。

## セットアップ

```
$ npm install
```

## 実行

```
$ npm run dev
```

## Tailwind CSS v4 の設定について

Tailwind CSS v4 では、設定は CSS ファイル内で行います。`tailwind.config.js` や `postcss.config.js` は不要です。

```css
@import "tailwindcss";
@import "@digital-go-jp/tailwind-theme-plugin/v4";
```

詳細は [src/index.css](./src/index.css) を参照してください。
